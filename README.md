# CarbonYield 🌍

**Symbolic Climate Contribution Platform** — fund verified carbon offset projects, earn symbolic credits, and reinvest your impact transparently.

### 🔗 Live Demo
[carbonyield.netlify.app](https://carbonyield.netlify.app)

### 🛠 Tech Stack
- React + Vite
- TailwindCSS v4
- React Router DOM
- Recharts, Mapbox
